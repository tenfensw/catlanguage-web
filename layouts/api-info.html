API reference

This service does have its own web API. I'm not sure why you would you use it in your app, but, maybe for entertainment purposes, it will be cool to use it.

Available functions (all of them accept only POST requests)

  • /do-meow - translates the text into Cat Language. Returns a single string with the translation of the text in plain text format.
  • /do-english - translate back to English. Returns a single string containing the original English text.

Examples (in Tcl)

package require http
package require tls
package require base64

http::register https 443 tls::socket ;# use TLS
set url "https://catlanguage.herokuapp.com/do-cat"
set token [http::geturl $url -query {Hey}]
set status [http::status $token]
set answer [http::data $token]
http::cleanup $token
http::unregister https

if {$status == {ok}} {
	puts $answer
}
Output (if everything goes ok): 
meoow meoow, meew, mow meew
